5. JavaScript

JavaScript is what makes a webpage do things. HTML gives it structure, CSS makes it look good, but JavaScript makes it interactive. Things like a button that responds when clicked, a form that validates your input, or content that updates without reloading — that is all JavaScript.

Here is a simple unit converter I built to demonstrate the basics. It reads a value the user types in, runs a calculation, and writes the result back onto the page without any page reload: 

function convertUnit() {
  var type  = document.getElementById("type").value;
  var value = parseFloat(document.getElementById("amount").value);
  var result = "";

  if (isNaN(value)) {
    result = "Please enter a number.";
  } else if (type === "kgLbs") {
    result = value + " kg  =  " + (value * 2.20462).toFixed(2) + " lbs";
  } else if (type === "kmMi") {
    result = value + " km  =  " + (value * 0.621371).toFixed(2) + " miles";
  } else if (type === "cF") {
    result = value + " °C  =  " + (value * 9/5 + 32).toFixed(2) + " °F";
  } else if (type === "mCm") {
    result = value + " m  =  " + (value * 100).toFixed(0) + " cm";
  }

  document.getElementById("output").innerText = result;
}

getElementById finds the input on the page and reads what was typed. The function works out the answer and uses innerText to write it back into the output paragraph. This is the core pattern JavaScript uses — read something, do something with it, show the result.
