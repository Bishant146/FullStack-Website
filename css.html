3. CSS — Cascading Style Sheets

CSS controls how a webpage looks. If HTML is the skeleton then CSS is the clothes — it handles colours, fonts, spacing and layout. Without CSS a webpage is just plain black text on a white background.

You write CSS by targeting an HTML element and setting properties on it. Here is a simple example that styles a heading and a button: 

h1 {
  color: #3d6b63;
  font-size: 24px;
  text-align: center;
}

button {
  background-color: #4a7a70;
  color: white;
  padding: 10px 22px;
  border: none;
  border-radius: 6px;
  font-size: 15px;
  cursor: pointer;
}

button:hover {
  background-color: #2d4d47;
}

The :hover rule only fires when the mouse moves over the element — making the page feel interactive without needing JavaScript. CSS also supports media queries which change the layout depending on the screen size, which is how sites look good on both desktop and mobile.
