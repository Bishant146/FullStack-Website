4. Bootstrap Framework

Bootstrap is a free CSS framework you can add to any webpage. Instead of writing all your own CSS from scratch it gives you ready-made classes to use. It saves a lot of time, especially when building layouts that need to work on different screen sizes.

The most useful thing I found is the grid system. Bootstrap divides the page into 12 columns and you decide how wide each section is. If you want three equal columns you use col-md-4 on each one — because 4 + 4 + 4 = 12. 

<!-- Link Bootstrap at the top of your HTML -->
<link rel="stylesheet"
  href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css">

<div class="container">
  <div class="row">
    <div class="col-md-4">Column 1</div>
    <div class="col-md-4">Column 2</div>
    <div class="col-md-4">Column 3</div>
  </div>
</div>

On a small screen Bootstrap automatically stacks those columns on top of each other — no extra CSS needed. It also comes with ready-made components that you can paste straight into a project.
