2. HTML — HyperText Markup Language

HTML is the foundation of every webpage. It gives the page its structure and content — headings, paragraphs, images, links and forms. Without it the browser has nothing to display. I like to think of it as the skeleton of a website because everything else gets built on top.

HTML works by wrapping content inside tags. The browser reads those tags and knows how to display each piece. Here is a basic example of a full HTML page: 

<!DOCTYPE html>
<html>
  <head>
    <title>My Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>This is my first webpage.</p>
    <a href="about.html">About Me</a>
  </body>
</html>

The <h1> tag creates a large heading, <p> makes a paragraph and <a> makes a clickable link. These are the most common tags you use when starting out.

HTML5 also added semantic tags like <header>, <nav>, <main>, <article> and <footer>. These are better than using <div> everywhere because they tell the browser exactly what the content is — which also helps with accessibility and search engines.
